# CareConnect — Healthcare Support Web App

A static landing page for a healthcare NGO platform that connects patients with volunteers and support resources. The project demonstrates a polished UI with registration tabs, a chatbot-style FAQ assistant, and a responsive layout.

## Features

- Responsive hero section with key stats
- Tabbed registration forms for patients, volunteers, and general contact
- AI FAQ assistant interface with quick reply buttons
- "How It Works" process section and sticky navigation
- Smooth scroll/animation effects for a modern user experience

## Project Structure

- `index.html` — main web page content and layout
- `Style.css` — reusable stylesheet for the layout and design
- `Script.js` — JavaScript behavior for tab switching, form interactions, scrolling effects, and chatbot handling

## Usage

1. Open `index.html` in a web browser.
2. Optionally use a local web server for a more realistic test environment.

## Notes

- This is a static HTML/CSS/JavaScript project.
- The page is built for local viewing and does not include a backend.
